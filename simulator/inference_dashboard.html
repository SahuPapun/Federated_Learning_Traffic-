ℹ️ How to Use

  1. Run training: python run.py
  2. Start API: python api_inference_server.py
  3. Upload a CSV with traffic values
  4. Select a client model
  5. Click Upload & Predict
  6. Explore charts & download results

CSV format: one column of numeric traffic values (e.g., source,LOSAng). The dashboard auto-detects the first numeric column.
